# Buffalo Crime Data Analysis & Predictive Modeling

This project presents an end-to-end machine learning analysis of the Buffalo Crime Dataset, focusing on data preprocessing, feature engineering, and predictive modeling to understand crime patterns and improve classification performance.

---

## Project Structure

- **buffalo_crime_analysis_ml.ipynb**  
  Contains the complete workflow including:
  - Data loading and preprocessing
  - Missing value handling
  - Feature engineering and encoding
  - Exploratory Data Analysis (EDA)
  - Correlation analysis
  - Machine learning model training
  - Neural network implementation
  - Performance evaluation and comparison

---

## Dataset Overview

The Buffalo Crime dataset contains real-world crime records including categorical and numerical features related to location, time, and offense type.

Preprocessing steps included:
- Handling missing values
- Encoding categorical variables
- Feature normalization
- Correlation-based feature selection
- Train/Validation/Test split

---

## Models Implemented

- Logistic Regression
- Random Forest 
- Neural Network

Models were evaluated using:
- Accuracy
- Confusion Matrix

---

## Key Insights

- Temporal and location-based features significantly influenced prediction performance.
- Feature encoding improved model interpretability.
- Neural networks demonstrated competitive performance compared to traditional ML models.
